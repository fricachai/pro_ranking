# 上市股票 ETF 籌碼與外資持股專業選股

此專案提供可重跑的台灣上市股票篩選程式與互動報告，整合：

- ETF 1／3／5／10／20日持股變化
- 主動ETF與被動ETF籌碼
- 外資實際持股1／5／10／20日變化
- 外資與三大法人買賣超
- 月營收、EPS、營業利益率、財務結構與估值
- EMA、RSI、MACD、乖離、波動與事件風險

## 查看報告

直接開啟根目錄的 `index.html`。股票名稱可另開 Yahoo 股市技術分析頁；完整排名表提供左右箭頭、上下兩組水平捲動方式，並固定排名與股票欄。

## 重新產生

需要 Node.js 18 以上版本：

```powershell
node .\full-professional-stock-screen.js
```

輸出會寫入 `professional-screen-report`：

- `latest.html`：最新互動報告
- `latest.json`：完整分析資料
- `full-professional-ranking-YYYYMMDD.csv`：完整排名

## 資料邊界

目前只篩選 ETF 資料集內持有的上市股票，排除上櫃股票。ETF來源可能標示資料不完整或部分ETF更新落後，因此20日資料只作背景，10日確認延續，5日用於判斷轉折。外資持股變化也可能受借券、海外存託憑證、股本異動與非市場交易影響，不能直接等同外資買賣超。

篩選結果是研究排序與風險檢查，不是保證報酬或個人化投資建議。
